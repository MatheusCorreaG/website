# Clínica Médica
## Descrição
O projeto da Clínica Médica é um website desenvolvido para uma clínica médica fictícia, fornecendo uma plataforma abrangente para gerenciar consultas médicas, pacientes e informações relacionadas.

Este site é uma solução completa que incorpora as seguintes funcionalidades:

Persistência em Banco de Dados: Utiliza um banco de dados para armazenar informações importantes, como detalhes dos pacientes, agendamentos de consultas e registros médicos.

Verificações de Login: Implementa um sistema de login seguro para médicos, funcionários e administradores, garantindo que apenas usuários autorizados possam acessar áreas específicas do site.

Agendamento de Consultas: Permite que os pacientes solicitem e agendem consultas médicas de acordo com a disponibilidade dos médicos e a agenda da clínica.

Listagens de Cadastros: Fornece listagens detalhadas de pacientes, médicos e funcionários cadastrados na clínica, facilitando a gestão e organização dos registros.

Requisições AJAX: Utiliza requisições AJAX para melhorar a experiência do usuário, permitindo atualizações de página dinâmicas e interativas sem recarregar a página inteira.

## Estrutura do repositório
O repositório está organizado da seguinte forma:

css: Contem todo o código CSS realizado no website
galeria: imagens utilizadas para a galeria do projeto
logo: local para armazenar a logo em diferentes resoluções
scriptSQL: Local para armazenar os scripts em SQL utilizados para a criação das tabelas
